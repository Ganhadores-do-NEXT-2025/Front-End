
# 🌐 Projeto Saúde Digital - Challenge FIAP 2025

## 🧠 Objetivo
Criar uma aplicação web intuitiva, acessível e responsiva para o Hospital das Clínicas – USP, focando na inclusão digital e na redução da taxa de absenteísmo em consultas médicas.

## 📁 Estrutura de Pastas
- `index.html` — Página principal
- `assets/` — Páginas auxiliares: FAQ, Cadastro, Cancelamento, etc.
- `css/` — Estilização da aplicação
- `js/` — Scripts de interatividade
- `README.md` — Descritivo técnico do projeto

## 💡 Tecnologias Utilizadas
- **HTML5**: Estrutura semântica e acessível
- **CSS3**: Design responsivo com media queries
- **JavaScript Vanilla**: Validação de formulários e interatividade
- **Sem frameworks ou bibliotecas externas**

## 👨‍💻 Funcionalidades
- Interface responsiva com suporte a dispositivos móveis, tablets e desktops
- Navegação simples com menus acessíveis
- Validação de formulários em JavaScript
- Páginas obrigatórias entregues conforme critérios do Challenge
